The HTML style attribute is used to apply CSS styles directly to HTML elements. Styles such as color, font, size, and alignment can be defined.

1. Style Attribute Syntax
The syntax for the style attribute is:
<tagname style="property:value;">
The property is a CSS property, and the value is the CSS value.

2. Background Color
The background-color property sets the background color for an element.
<body style="background-color:powderblue;">
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>

You can set different background colors for multiple elements:
<h1 style="background-color:powderblue;">Heading</h1>
<p style="background-color:tomato;">Paragraph</p>

3. Text Color
The color property sets the text color of an element.
<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

4. Fonts
The font-family property sets the font type for an element.
<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

5. Text Size
The font-size property defines the size of text.
<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

6. Text Alignment
The text-align property sets the horizontal alignment of text.
<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>


Chapter Summary:
Use the style attribute to add inline styles to elements.
background-color sets background colors.
color sets text color.
font-family sets the font.
font-size adjusts text size.
text-align controls text alignment.
